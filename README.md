# Andmekirjelduse standardi repositoorium

See repositoorium on loodud eesmärgiga dokumenteerida ja kirjeldada andmekirjelduse standardit ning sisaldab selle ametlikku versioonihaldust.

Uues veebikeskkonnas asuv andmekirjelduse standard: https://e-gov.github.io/Andmekirjelduse-Standard/

PDF kujul versioon 3.0.1 saadaval siin: 
https://stat.ee/sites/default/files/2024-08/AH_juhis_andmekirjeldus_standard_3.0.1.pdf

## Versioonid

Versioonid on tähistatud Git tag'iga.

## Ettepanekud muudatusteks

Parandused ja muudatusettepanekud esitada GitHub issues kaudu.
